# List of Symbols

## Latin Letters

| Symbol | Description |
|--------|-------------|
| A | Helmholtz energy |
| A_ij | Activity coefficient model parameters |
| a | Cubic equation parameter |
| a_T | Derivative of a with respect to T |
| B | Cubic equation parameter (dimensionless) |
| b | Cubic equation parameter |
| C | Cubic form |
| Cp_0 | Ideal gas heat capacity at constant pressure |
| F | Poynting correction factor |
| f_i | Fugacity of species i in solution |
| G | Gibbs free energy |
| G^E | Excess Gibbs free energy |
| H | Enthalpy |
| H^R | Residual enthalpy |
| H^E | Excess enthalpy |
| I | Auxiliary variable in residual property calculations |
| K_i | VLE equilibrium constant of component i |
| k_1 | Cubic equation parameter |
| k_2 | Cubic equation parameter |
| k_3 | Cubic equation parameter |
| k_ij | Binary interaction parameter |
| M | Specific or molar value of an extensive thermodynamic property |
| M^R | Residual property |
| n | Number of moles |
| P | Absolute pressure |
| Pc | Critical pressure |
| Pr | Reduced pressure |
| Q | Matrix in quadratic form / Heat |
| q_ij | Elements of **Q** |
| R | Universal gas constant |
| S | Entropy |
| S^R | Residual entropy |
| S^E | Excess entropy |
| T | Absolute temperature |
| Tc | Critical temperature |
| Tr | Reduced temperature |
| V | Volume |
| v | Specific volume |
| w | Acentric factor |
| x | Liquid molar fraction |
| y | Vapor molar fraction |
| Z | Compressibility factor |

## Greek Letters

| Symbol | Description |
|--------|-------------|
| alpha | Cubic equation parameter / Activity |
| Delta | Discriminant |
| phi | Pure component fugacity coefficient |
| phi_i | Fugacity coefficient of component i in solution |
| Omega_a | Cubic equation constant |
| Omega_b | Cubic equation constant |
| gamma | Activity coefficient |
| Lambda | Wilson model parameter |
| delta | Solubility parameter |
| mu | Chemical potential |

## Superscripts

| Symbol | Description |
|--------|-------------|
| gi | Ideal gas |
| E | Excess property |
| l | Liquid phase |
| R | Residual property |
| sat | Saturation condition |
| SI | Ideal solution |
| v | Vapor phase |
| 0 | Reference state |

## Subscripts

| Symbol | Description |
|--------|-------------|
| fg | Phase change (vapor to liquid) |
| i, j, n | Component indices |
| r | Reduced condition |
| ref | Reference state |
| 0 | Test point (initial state) |
